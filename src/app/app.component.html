What's this?
Use this simple (yet complex) tool to quickly check if a name is (most likely) Muslim or not. I did this to get myself up to speed with Angular and Typescript. Here's the sourcecode (GPL). It's a good sample of a standard Angular Web App. It has routes, unit tests, lazyloads, etc, all done the Angular way!
What's this for?
Impress your Muslim customers, friends or coworkers, by dropping a 'Salam', 'Salamualaykum' or 'Ramadhan Mubarak' in your communications! Off course, use this tool to check first if they are one!
What's the catch?
Nothing, this is Free! Just buy me a coffee if you like to support this project
Is this accurate?
So far it's been pretty good! I am from Australia where the Muslim community is diverse => Arabs, Africans, Balkans, Asians etc; Having said that, it wasn't easy setting this up. Had to take care of special cultural names and phonetics. I tried using Machine Learning (AI), but couldn't get it to work. Would be great if you can help me with that.
